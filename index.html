Majority of us live in a busy lifestyle where we tend to stick to our routines. Generally we will also try to re-prioritise our routine each day so that we can be more efficient.

In that I'll share with you an example of what my routine is like as a husband and a father of two boys:
- My morning is usually occupied ensuring that my eldest son are well fed in the morning, all dressed up ready for school and not to mention ensuring that my eldest son'slunchbox and crunch and sip are ready to start the day. Once I palm off the responsibility to the school, that’s when I can leave for work.

- After work, I will fetch the kids home and ensure that dinner is ready on time for the family, so I can get the kids all sorted ready for bed. Once the kids are asleep this is when I get to do whatever I want - literally.

- This is generally my daily routine in brief with a slight difference on weekends with more chores to do - but you get the idea without me blabbering too much.

- Unfortunately, my routine is non negotiable as I report to three bosses at home. This is the story of my life.

Hi Sally!

virtual assistant powered by voice personalised

python speech to text wakeword

speech synthesis API text to speech

This is where I found Sally, a side project which I built in the attempt to address some of what I have mentioned.

She is a voice virtual assistant, built in Python and is currently hooked up to:
- Google Speech API for transcribing speech to text
- eSpeak for speech synthesis enablement
- Hotword or wakeword integration which provides a more personalised experience
- Last but not least, integrated with purposeful APIs - and at this point it’s only with a weather API

There is definitely a lot more to enhance with Sally, but I’m hoping that once I release the first version of Sally publicly, I get to encourage contributors who are keen learners to contribute this project as a learning experience

If you are wondering if this is built to replace Amazon Alexa, Google Assistant or the likes - the answer is no. This is merely a side project for myself to learn and explore the capability of voice user interface
